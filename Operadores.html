Ordem de precedência dos operadores

1° - Operações Aritmeticas

Sequência: ' () ' ; ' ** ' ; ' / e * ' ; ' + e - ' .

2° - Operações Relacionais

Sequência: Não há.

3° - Operações Lógicos

Sequência: ' ! ' , ' && ' , ' || ' .

Operador Ternário

Teste ? True : False

No teste é qualquer teste que retornara um booleano. Após o '?' é o que acontecera se for verdadeiro. E depois do ':' é o que acontecera se for falso.

O operador térnario também éo ultimo na ordem de precedência
